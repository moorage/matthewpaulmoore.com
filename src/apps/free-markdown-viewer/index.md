Free Markdown Viewer is a simple, lightweight, local-first app for opening folders full of Markdown files and browsing them across macOS, iPhone, and iPad.

Use it to read notes, documentation, knowledge bases, exported reports, and project folders without setting up an account or relying on a web service.

## Why it is useful

- Open a folder and browse Markdown files from a sidebar
- Read Markdown with native rendering for headings, lists, tables, code blocks, images, animated images, and local video
- Navigate quickly between files with back and forward history
- Adjust viewer font size for comfortable reading
- Work entirely with files you choose from Finder or the Files app

## Why it is different

The renderer stays native. There is no `WKWebView`, HTML, CSS, or JavaScript rendering layer between you and your documents.

Free Markdown Viewer is also FOSS. The source is available at <https://github.com/moorage/free-markdown-viewer>.

It is designed to stay direct:

- no account required
- no subscription
- no in-app purchase
- no tracking
